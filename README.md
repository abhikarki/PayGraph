## PayGraph - A Multi-Rail Payment Routing Engine 
PayGraph is a payment orchestration system that models global settlement networks (bank transfers, card rails, and stablecoin infrastructure) as a dynamic graph to compute optimal transaction routes.
The system explores how modern fintech platforms can optimize cross-network payments using multi-objective routing algorithms, real-time network telemetry, and graph-driven visualization.

Global money movement today involves fragmented settlement networks like traditional banking rails, card networks, local payout systems, and stablecoin/blockchain settlement.
Selecting the optimal transaction path requires balancing **cost, latency, reliablility, and liquidity constraints**.
PayGraph investigates this problem from a systems and infrastructure perspective, inspired by large-scale payment orchestration platforms.
